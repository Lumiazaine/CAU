====== Acceso a Diraya ======

Se accede a través del Escritorio Judicial con el enlace Consultas Diraya

{{:pag_apli:diraya:acceso_diraya.png?600|}}}}

Una vez introducidos los datos obligatorios que se solicitan en la pantalla anterior, se abre la URL http://hsu.sas.junta-andalucia.es/hsd_unica/VisorHsdu.aspx que depende del SAS.

Resolución de incidencias: Se debe usar preferentemente Edge con compatibilidad a IE y hacer la limpieza procedimentada si existen problemas de acceso. Si las incidencias se refieren a la web de Diraya, es decir, en la URL  http://hsu.sas.junta-andalucia.es/, deben crear incidencia en el SAS.
