====== Configuración Hardware Integrity ======

[[https://xwiki.justicia.junta-andalucia.es/xwiki/bin/edit/Sistemas/HP INTEGRITY BL870C (NPAR,VPAR y Cluster Service Guard)?section=4|Editar]]

Disponemos de 4 nPares. Cada nPar se componen de dos particiones físicas unidas por un front panel.

Cada partición física tiene los siguientes componentes hardware:

  * placa base independiente
  * CPU Intel(R) Itanium(R) Processor 8-core 2.53 GHz with 32MB Last Level Cache 9560 (8 Cores)
  * 192GB de RAM
  * varios discos locales (solo se utilizan para el arranque uEFI, luego se continua el arranque con los discos de cabina)
  * interfaces de red
  * interfaces de fibra (hba) 10Gigabit.

Por lo que cada nPar dispone del doble de los componentes anteriores, es decir:

  * 2 placa base
  * 2 CPUs Intel(R) Itanium(R) Processor 8-core 2.53 GHz with 32MB Last Level Cache 9560 (8 Cores). Esto supone un total de 16 cores por nPar.
  * 384GB de RAM
  * varios discos locales (solo se utilizan para el arranque uEFI, luego se continua el arranque con los discos de cabina)
  * interfaces de red
  * interfaces de fibra (hba) 10Gigabit.

NOTA: Se ha realizado una ampliación de hardware para dos de las cuchillas (Bay 5,6 y 7,8 que corresponde a iusvsp01 y iusvsp04, dejandolas con 32 CPUs a 2527 Mhz y 575.8 GB de RAM
