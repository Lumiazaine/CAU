Tareas Programadas de GTAB

Introducción

Todos los scripts están en /root/scripts/nsr y mediante el comando “crontab -l” podemos ver las diferentes tareas.

crontab

* * * * * comando

Clonaciones

Ubicacion de los scripts: /root/scripts/nsr/clonado

El clonado a cintas se realiza en el servidor ORAGRID mediante scripts desarrollados por Patricio Mora

Todas envían un correo cuando finaliza al correo “notificaciones.gtab.ius@juntadeandalucia.es”

==== BBDD ====

Tarea: clonado-semanal-BBDD-viernes (X)

Fecha: Todos los Sábados a las 19:13

==== ADRIANO ====

Tarea: clonado-semanal-ADRIANO (FULL)

Fecha: Todos los Lunes a las 3:13

==== COMUNES ====

Tarea: clonado-trimestral-COMUNES

Fecha: Cada 3 meses a las 3:13 el día 3

==== Otros ====

Este script vigila que haya suficiente espacio en cinta disponible para clonado de BBDD y ADRIANO, si no hay espacio manda un correo avisandolo.

Tarea: cintasClonadoDisponibles

Fecha: A las 12:44 los jueves y viernes
