PLANTILLAS GENÉRICAS Y PARA LA GESTIÓN DEL PROYECTO

A continuación se encuentran las plantillas relativas a la gestión del proyecto.

Código Entregable Descripción Plantilla
ACT Acta de entrega
Se incluye la plantilla para elaborar el acta de una entrega (alcance, tareas realizadas y consideraciones). 		Acta de entrega
TD Acta de reunión
Se incluye una plantilla para elaborar el acta de una reunión. 		Acta de reunión
AGR Agenda de reunión
Se incluye la plantilla para elaborar un informe de convocatoria de una reunión. 		Agenda de reunión
IRP Informe de resultados y cierre de proyecto
Se incluye la plantilla para elaborar un informe de resultados y de cierre de proyecto. 		Informe de resultados y cierre de proyecto
ISP Informe de seguimiento de proyecto
Se incluye la plantilla para elaborar un informe de seguimiento del proyecto. 		Informe de seguimiento de proyecto
PAR Plan de acción de riesgos
Se incluye la plantilla para generar un plan de acción de riesgos. 		Plan de acción de riesgos
PCR Petición de cambio
Se incluye la plantilla para la solicitud de algún cambio dentro del proyecto. 		Petición de cambio
PT Programa de trabajo
Se incluye una plantilla para elaborar un plan de trabajo. 		Programa de trabajo

A continuación se encuentran otras plantillas de interés.

Código Entregable Descripción Plantilla
ENF Encuesta de formación
Se incluye una plantilla de encuesta para las formaciones. 		Encuesta de formación
HFF Hoja de firmas de formación
Se incluye la plantilla de las hojas de firma para la formación. 		Hoja de firmas de formación
IIN Informe de incurridos
Se incluye la plantilla para elaborar el informe de los incurridos. 		Informe de incurridos
INF Informe de formación
Se incluye la plantilla para elaborar el informe de la formación. 		Informe de formación
PLF Plan de formación
Se incluye el documento con el plan de la formación. 		Plan de formación
