El Editor

Para acceder al modo de edición de cualquier página, debemos dirigirnos al icono "Editar esta página" (lápiz) que se encuentra en el menú de la derecha.

Una vez accedemos, el siguiente punto importante es asegurarnos de que estamos en el editor correcto.

Debemos trabajar con el editor DW y NO con el editor CKG.

Para cambiar entre ellos pulsaremos en el siguiente botón:

IMPORTANTE: El cambio entre editores puede dar lugar a que la estructura de la página en la que nos encontramos se "rompa".

Siempre que realicemos un cambio entre editores, DokuWiki nos sacará del modo edición automáticamente, este es el momento de comprobar que la página se encuentre en correcto estado y no haya sufrido errores.

Si no existen errores, podemos volver a acceder al modo edición y comenzar a editar el contenido.

Si existen errores, debemos acceder al icono "Revisiones antiguas" (reloj).

Una vez dentro, seleccionaremos la última versión antes de que accediéramos y podremos visualizarla.

Cuando comprobemos que esa es la versión que queremos recuperar, pulsaremos sobre el icono "Restaurar" (flecha circular) para confirmar.

Finalmente, podemos volver a acceder al modo edición, en el que se abrirá por defecto el editor DW y podemos comenzar a editar el contenido.

Etiquetas Básicas en Sintaxis DokuWiki

Lo primero y muy importante que debemos tener en cuenta es que todas las etiquetas que utilicemos, deben ir debidamente cerradas. Por ejemplo:

  • Abrir etiqueta → <WRAP>
  • Cerrar etiqueta → </WRAP>

Todo el contenido que quede entre la apertura y cierre de una etiqueta, se verá afectado por esta.

A continuación vamos a ver las etiquetas que más vamos a utilizar en DokuWiki y cómo funcionan.

           

<WRAP>

La etiqueta WRAP sirve para meter contenido dentro de un “contenedor” al que puedes aplicar formato.

Este contenedor por si sólo no es visible, debemos aplicarle alguna etiqueta de estilo como por ejemplo:

  • <WRAP centeralign / leftalign / rightalign> → posiciona el texto centrado / izquierda / derecha

  • <WRAP box> → crea un contenedor gris

  • <WRAP box centeralign> → combina ambos estilos

  • <WRAP info round> → info indica el tipo de panel y round indica que queremos que este sea redondeado

  • <WRAP tip>

  • <WRAP important>

           

<panel>

Al contrario que WRAP, esta etiqueta mostrará el contenido en un “contenedor” que sí es visible por si solo y al que también podremos añadir estilos.

  • <panel>

  • <panel title="TÍTULO">

  • <panel title="TÍTULO" type="info">

  • <panel title="TÍTULO" type="success">

  • <panel title="TÍTULO" type="warning">

  • <panel title="TÍTULO" type="danger">

Vamos a ver ejemplos del uso de estas etiquetas en DokuWiki anidando paneles:

  • Panel anidado dentro del panel principal:
    • Estructura:

    <panel title="PANEL PRINCIPAL">

    Texto del panel principal.

    <WRAP>

    <panel title="PRIMER PANEL INTERIOR">

    Contenido del primer panel interior.

    </panel>

    </WRAP>

    </panel>

Herramientas del Editor

En el modo edición de DokuWiki podemos encontrar una barra con distintas herramientas útiles para dar formato y añadir otros tipos de contenido tales como enlaces o imágenes.

Vamos a ver cuáles son y a su vez como usarlas de forma manual, sin necesidad de usar la barra de herramientas.

           

Estilos de Texto

  • Negrita:

    • → **HOLA** → HOLA

  • Cursiva:

    • → //HOLA// → HOLA

  • Subrayado:

    • → __HOLA__ → HOLA

  • Código:

    • → ''HOLA'' → HOLA

  • Tachado:

    • → <del>HOLA</del> → HOLA

           

Títulos

  • → ====== TÍTULO 1 ====== →

    TÍTULO 1

  • → ===== TÍTULO 2 ===== →

    TÍTULO 2

  • → ==== TÍTULO 3 ==== →

    TÍTULO 3

  • → === TÍTULO 4 === →

    TÍTULO 4

  • → == TÍTULO 5 == →
    TÍTULO 5
         

Insertar Enlaces, Documentos e Imágenes

DokuWiki nos permite insertar tanto enlaces internos y externos como incrustar imágenes y documentos.

De nuevo, vamos a ver como usar una por una desde la barra de herramientas y de forma manual.

  • Enlace interno:

    • Este formato sólo puede utilizarse para enlazar páginas de DokuWiki.
    • Lo primero es saber el nombre interno de la página de DokuWiki a la que queremos que el enlace nos redirija, la buscaremos externamente en una nueva pestaña o ventana, donde podremos comprobar dicho nombre.

    • De vuelta al editor, pulsaremos el icono "Enlace interno" el cuál nos abrirá una nueva ventana en la que buscaremos la página.

    • Observaremos que en el editor aparecerá, con nuestro ejemplo → [[pag_puesto|]]. En este punto tenemos dos opciones:
      1. No añadir ningún nombre al enlace, de forma que el resultado visual será un enlace con el nombre interno de la página: [[pag_puesto|]] → pag_puesto
      2. Añadir tras el icono | el nombre con el que queremos que se visualice el enlace: [[pag_puesto|Puesto de Trabajo]] → Puesto de Trabajo

  • Enlace externo:

    • Este formato puede utilizarse tanto para enlazar páginas de DokuWiki como páginas externas.
    • En este caso necesitaremos la URL de la página a la que queremos que nos redirija el enlace.

    • De vuelta al editor, pulsaremos el icono "Enlace externo" el cuál nos arrojará en el editor lo siguiente → [[http://example.com|Enlace externo]]. De nuevo disponemos de dos opciones:
      1. No añadir ningún nombre al enlace, sustituiremos el primer apartado http://example.com por el enlace deseado y eliminaremos el segundo apartado Enlace externo dejándolo en blanco de forma que el resultado visual será el propio enlace: [[https://extranet.chap.junta-andalucia.es/dokuwiki/!ssdj/doku.php?id=pag_puesto|]] → https://extranet.chap.junta-andalucia.es/dokuwiki/!ssdj/doku.php?id=pag_puesto
      2. Añadir tras el icono | el nombre con el que queremos que se visualice el enlace: [[https://extranet.chap.junta-andalucia.es/dokuwiki/!ssdj/doku.php?id=pag_puesto|Puesto de Trabajo]] → Puesto de Trabajo
