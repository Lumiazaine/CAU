Procedimientos de Instalación y Configuración de Dispositivos de Red

1. Objetivo

Establecer los pasos y buenas prácticas para la correcta instalación y configuración inicial de dispositivos de red tales como switches, routers, puntos de acceso, firewalls y balanceadores, garantizando seguridad, eficiencia operativa y trazabilidad.

2. Alcance

Este procedimiento se aplica a todos los dispositivos de red gestionados por el Departamento de Redes en las diferentes sedes y CPDs, tanto en entornos de producción como en proyectos de renovación o expansión.

3. Responsables

  • Técnicos del área de Redes y Comunicaciones
  • Administradores de Seguridad (en configuraciones críticas)

4. Procedimiento Paso a Paso

4.1. Preparación previa

  1. Verificar el plan de direccionamiento IP.
  2. Identificar VLANs y roles del dispositivo.
  3. Consultar configuración estándar o plantilla de base.
  4. Confirmar versión de firmware recomendada.

4.2. Instalación física

  1. Montaje en rack si aplica (cumpliendo normativas de cableado y ventilación).
  2. Alimentación eléctrica redundante (si es compatible).
  3. Conexiones uplink verificadas (troncales o puertos de distribución).

4.3. Configuración básica inicial

  1. Asignación de IP de gestión.
  2. Definición de nombre de host.
  3. Habilitación de acceso seguro: SSH / HTTPS.
  4. Configuración de usuarios y credenciales seguros.
  5. Aplicación de ACLs de acceso administrativo.

4.4. Configuración funcional

  1. Definición de VLANs.
  2. Enrutamiento estático o dinámico según el rol.
  3. Políticas QoS si aplican.
  4. Enlace a autenticación externa (RADIUS/LDAP/802.1X).

4.5. Verificación y pruebas

  1. Conectividad local y remota.
  2. Pruebas de rendimiento básico (ping, throughput).
  3. Prueba de acceso administrativo desde subred autorizada.
  4. Registro de logs de arranque y eventos clave.

4.6. Documentación y traspaso

  1. Registro de configuración inicial y backup.
  2. Actualización del inventario de red.
  3. Notificación de disponibilidad a los responsables de servicios.

5. Seguridad y Recomendaciones Técnicas

  1. No dejar interfaces de administración expuestas en redes de usuario.
  2. Cambiar las contraseñas predeterminadas antes de conectar el equipo a red.
  3. Aplicar las políticas de bastionado definidas en bastionado.
  4. Habilitar syslog y SNMPv3 para monitoreo centralizado.

6. Registro de Cambios y Control

  1. Las configuraciones deben almacenarse cifradas y con control de versiones.
  2. Toda instalación debe estar asociada a una solicitud o ticket de trabajo.
