Módulo antifraude

La funcionalidad Antifraude establece un mecanismo de control en la asignación de las citas como medida preventiva para el agotamiento o saturación fraudulenta en las asignaciones de citas previas.

Con esta funcionalidad el sistema controlará la emisión de tickets mediante la identificación de documento de identidad, de modo que un ciudadano podrá solicitar una cita día por día para cada trámite ofertado en cada Registro Civil.
