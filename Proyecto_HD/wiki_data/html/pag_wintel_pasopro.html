Procedimiento de Paso a Producción – Máquinas Virtuales y Configuraciones

1. Objetivo

Definir los pasos necesarios para pasar a producción máquinas virtuales o cambios de configuración, asegurando calidad, seguridad y trazabilidad.

2. Roles

  • Solicitante: Registra y justifica el cambio.
  • Administrador de Sistemas: Ejecuta el despliegue y pruebas.
  • Responsable de TI: Valida y autoriza el cambio.
  • Comité de Cambios (CAB): Aprueba el paso definitivo a producción.

3. Requisitos Previos

  1. Solicitud registrada con plan de implementación y reversión.
  2. Pruebas superadas en entorno de pre-producción.
  3. Backup verificado antes de la ejecución.
  4. Plan de comunicación a usuarios y equipos técnicos.

4. Procedimiento

4.1 Preparación

  1. Confirmar recursos y compatibilidad técnica.
  2. Configurar y validar la VM en entorno preproductivo.
  3. Documentar configuración aplicada.

4.2 Paso a Producción

  1. Confirmar ventana de mantenimiento.
  2. Realizar snapshot o backup previo.
  3. Implementar la VM o configuración en producción.
  4. Validar funcionamiento y documentar incidencias.

4.3 Cierre

  1. Actualizar inventario de activos.
  2. Adjuntar evidencias al registro del cambio.
  3. Aprobación final por CAB.
  4. Activar monitorización.
