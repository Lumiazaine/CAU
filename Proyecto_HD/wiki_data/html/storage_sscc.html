ALMACENAMIENTO SSCC

Objetivo

En este documento se describe la infraestructura de almacenamiento de SSCC con el objeto defacilitar la gestión de los procesos y servicios asociados.

Introducción

SSCC dispone de diversa infraestructura de almacenamiento, con una finalidad variada. Se dispone de una cabina de almacenamiento HP3PAR que proporciona almacenamiento por fibra de datos fundamentalmente a los sistemas de virtualización y BBDD, existe una Isilon que proporciona servicios de NAS y disponemos de diversos Data Domain que proporcionan servicios de VTL. Existen otros dispositivos tales como librerías de cintas para salvaguardar el backup. Estos y otros elementos se describirán a lo largo del documento.

Alcance

Se tratará con el servicio de almacenamiento a SSCC, donde se ofrecen diferentes volúmenes que se presentan mediante LUNs, NFS o CIFS a los diferentes servidores.
