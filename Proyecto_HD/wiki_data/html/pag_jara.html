# 📄 JARA

## 1️⃣ Introducción 📝

La aplicación JARA automatiza dos funciones clave entre el Órgano Judicial y el Archivo de la Sede Judicial:

  • - Transferencias
  • - Servicio de préstamos

También gestiona prórrogas, envíos a la Junta de Expurgo y homogeneiza la gestión de datos del Archivo de la Sede Judicial. Es una aplicación web accesible a través del Escritorio Judicial para usuarios adscritos.
