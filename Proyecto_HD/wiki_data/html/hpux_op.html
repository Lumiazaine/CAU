Introducción

En este documento se van a describir las principales instrucciones técnicas asociadas a la administración de servidores HP-UX

Ampliación de volume groups en HP-UX

Editar Los siguientes pasos explican como ampliar un vg en un sistema operativo HP-UX.

* Ampliamos previamente en la 3PAR el VG al tamaño requerido. Lista los VG, LV y PV:
# vgdisplay -v

Refrescamos los cambios del VG que queramos ampliar
# Vgmodify -a -E “/vgxx” Extendemos el LV con el tamaño de PE que vamos a utilizar:
# lvextend -l 5222135 vgxx/datosxx

Le damos formato al vg extendido en Megabytes indicando la ruta, en este caso será en FS:
# fsadm -F vxfs -v 163776M /opt/oracle/oradata/bdpro01

Gestión de CLUSTERS

Editar Ver status del cluster

  • cmviewcl

Detener un package

Nota: al lanzar esta orden, el software del cluster deshabilita el AUTO_RUN del package, siendo necesario reactivarlo cuando se finalicen las operaciones.

  • cmhaltpkg package
  • cmhaltpkg -n NODO package

Arrancar un package

  • cmrunpkg package
  • cmrunpkg -n NODO package

Activar el AUTO_RUN

Nota: adicionalmente arranca automaticamente el package si este se encuentra detenido.

  • cmmodpkg -e package

Activar el SWITCHING

  • cmmodpkg -e -n NODO package

Detener un nodo del cluster

Nota: esto detiene el servicio de cluster en ese nodo hasta que se reinicie nuevamente el demonio, los cambios producidos en el cluster no se propagarán hasta que se reactive.

  • cmhaltnode -n NODO

Detener todo el cluster

  • cmhaltcl CLUSTER

NTP

Editar

Para el cambio de servidores de NTP editamos el fichero ntp.conf añadiendo/modificando al final de la línea los servidores.

# vi /etc/ntp.conf

Reiniciamos el servicio

# /sbin/init.d/xntpd stop && /sbin/init.d/xntpd start

Comprobamos que están sincronizando los servidores correctos.

# ntpq -c pee

# ntpq -c ass
